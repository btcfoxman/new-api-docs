# 获取 Kling 图生视频任务状态

- Source: `relay.json`
- Method: `GET`
- Path: `/kling/v1/videos/image2video/{task_id}`
- Operation ID: `getKlingImage2Video`

## 说明

查询 Kling 图生视频任务的状态和结果。

## 参数

| 名称 | 位置 | 必填 | 描述 |
|---|---|---|---|
| task_id | path | True | 任务 ID |

## 请求体

无

## 响应

| 状态码 | 说明 |
|---|---|
| 200 | 成功获取任务状态 |
| 404 | 任务不存在 |
