# Kling 文生视频

- Source: `openapi.json`
- Method: `POST`
- Path: `/kling/v1/videos/text2video`
- Operation ID: `createKlingText2Video`

## 说明

该系列支持OpenAI视频生成格式（详见链接 [视频生成/Sora兼容格式](/api-reference/cn/视频生成sora兼容格式/创建视频)）。

使用 Kling 模型从文本描述生成视频。

支持的模型：kling-v1, kling-v1-5 等

## 参数

无

## 请求体

```json
{'content': {'application/json': {'schema': {'$ref': '#/components/schemas/VideoRequest'}, 'example': {'model': 'kling-v1', 'prompt': '宇航员站起身走了', 'duration': 5, 'width': 1280, 'height': 720, 'fps': 30}}}}
```

## 响应

| 状态码 | 说明 |
|---|---|
| 200 | 成功创建视频生成任务 |
| 400 | 请求参数错误 |
