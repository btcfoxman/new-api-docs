# Kling 图生视频

- Source: `relay.json`
- Method: `POST`
- Path: `/kling/v1/videos/image2video`
- Operation ID: `createKlingImage2Video`

## 说明

使用 Kling 模型从图片生成视频。

支持通过 image 参数传入图片 URL 或 Base64 编码的图片数据。


## 参数

无

## 请求体

```json
{'content': {'application/json': {'schema': {'$ref': '#/components/schemas/VideoRequest'}, 'example': {'model': 'kling-v1', 'prompt': '人物转身走开', 'image': 'https://example.com/image.jpg', 'duration': 5, 'width': 1280, 'height': 720}}}}
```

## 响应

| 状态码 | 说明 |
|---|---|
| 200 | 成功创建视频生成任务 |
| 400 | 请求参数错误 |
