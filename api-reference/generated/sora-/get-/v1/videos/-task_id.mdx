# 获取视频任务状态 

- Source: `relay.json`
- Method: `GET`
- Path: `/v1/videos/{task_id}`
- Operation ID: `getVideo`

## 说明

OpenAI 兼容的视频任务状态查询接口。

返回视频任务的详细状态信息。


## 参数

| 名称 | 位置 | 必填 | 描述 |
|---|---|---|---|
| task_id | path | True | 视频任务 ID |

## 请求体

无

## 响应

| 状态码 | 说明 |
|---|---|
| 200 | 成功获取视频任务状态 |
| 404 | 任务不存在 |
