# 获取视频内容

- Source: `relay.json`
- Method: `GET`
- Path: `/v1/videos/{task_id}/content`
- Operation ID: `getVideoContent`

## 说明

获取已完成视频任务的视频文件内容。

此接口会代理返回视频文件流。


## 参数

| 名称 | 位置 | 必填 | 描述 |
|---|---|---|---|
| task_id | path | True | 视频任务 ID |

## 请求体

无

## 响应

| 状态码 | 说明 |
|---|---|
| 200 | 成功获取视频内容 |
| 404 | 视频不存在或未完成 |
