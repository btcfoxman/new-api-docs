# Gemini 图片(Nano Banana)

- Source: `openapi.json`
- Method: `POST`
- Path: `/v1beta/models/{model}:generateContent`
- Operation ID: `geminiRelayV1Beta`

## 说明

Gemini 图片生成。

Nano Banana系列支持OpenAI Dall-E格式（详见链接 [图片生成/OpenAI Dall-E格式](#tag/%E5%9B%BE%E7%89%87%E7%94%9F%E6%88%90%2FOpenAI%20Dall-E%E6%A0%BC%E5%BC%8F)）。

## 参数

| 名称 | 位置 | 必填 | 描述 |
|---|---|---|---|
| model | path | True | 模型名称 |
