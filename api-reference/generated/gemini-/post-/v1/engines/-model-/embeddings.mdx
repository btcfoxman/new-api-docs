# Gemini 嵌入(Embeddings)

- Source: `relay.json`
- Method: `POST`
- Path: `/v1/engines/{model}/embeddings`
- Operation ID: `createEngineEmbedding`

## 说明

使用指定引擎/模型创建嵌入

## 参数

| 名称 | 位置 | 必填 | 描述 |
|---|---|---|---|
| model | path | True | 模型/引擎 ID |

## 请求体

```json
{'content': {'application/json': {'schema': {'$ref': '#/components/schemas/EmbeddingRequest'}, 'examples': {}}}}
```

## 响应

| 状态码 | 说明 |
|---|---|
| 200 | 成功创建嵌入 |
