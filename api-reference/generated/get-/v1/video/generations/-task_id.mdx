# 获取视频生成任务状态

- Source: `relay.json`
- Method: `GET`
- Path: `/v1/video/generations/{task_id}`
- Operation ID: `getVideoGeneration`

## 说明

查询视频生成任务的状态和结果。

任务状态：
- `queued`: 排队中
- `in_progress`: 生成中
- `completed`: 已完成
- `failed`: 失败


## 参数

| 名称 | 位置 | 必填 | 描述 |
|---|---|---|---|
| task_id | path | True | 任务 ID |

## 请求体

无

## 响应

| 状态码 | 说明 |
|---|---|
| 200 | 成功获取任务状态 |
| 404 | 任务不存在 |
