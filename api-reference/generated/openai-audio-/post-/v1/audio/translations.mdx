# 音频翻译

- Source: `relay.json`
- Method: `POST`
- Path: `/v1/audio/translations`
- Operation ID: `createTranslation`

## 说明

将音频翻译为英文文本

## 参数

无

## 请求体

```json
{'content': {'multipart/form-data': {'schema': {'type': 'object', 'properties': {'file': {'type': 'string', 'format': 'binary', 'example': ''}, 'model': {'type': 'string', 'example': ''}, 'prompt': {'type': 'string', 'example': ''}, 'response_format': {'type': 'string', 'example': ''}, 'temperature': {'type': 'number', 'example': 0}}, 'required': ['file', 'model']}}}, 'required': True}
```

## 响应

| 状态码 | 说明 |
|---|---|
| 200 | 成功翻译 |
