# 创建视频生成任务（Veo3 Google原生格式，暂不支持）

- Source: `openapi.json`
- Method: `POST`
- Path: `/v1beta/models/{model}:predictLongRunning`
- Operation ID: `create_veo3_google_native_task`

## 说明

该系列支持OpenAI视频生成格式（详见链接 [视频生成/Sora兼容格式](/api-reference/cn/视频生成sora兼容格式/创建视频)）。

该接口专题页仅用于说明 Veo 原生格式与当前网关支持状态。

当前 Veo 系列对外请使用 OpenAI Sora 兼容模式：`POST /v1/videos`。

Google 原生模式路径预留为：`POST /v1beta/models/{model}:predictLongRunning`。当前网关尚未接通 Veo 原生任务分发链路，因此按原生 `instances/parameters` 请求体调用暂不支持。

## 参数

| 名称 | 位置 | 必填 | 描述 |
|---|---|---|---|
| model | path | True | Veo 模型名称。 |
