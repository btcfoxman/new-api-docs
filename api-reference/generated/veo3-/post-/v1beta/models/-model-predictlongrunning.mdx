# 创建视频生成任务（Veo3 Google原生格式）

- Source: `openapi.json`
- Method: `POST`
- Path: `/v1beta/models/{model}:predictLongRunning`
- Operation ID: `create_veo3_google_native_task`

## 说明

该系列支持OpenAI视频生成格式（详见链接 [视频生成/Sora兼容格式](/api-reference/cn/视频生成sora兼容格式/创建视频)）。

Google Veo 原生视频生成接口。请求格式为 application/json，创建成功后返回 `operation name`。

推荐两种接入方式：
- OpenAI Sora 兼容模式：`POST /v1/videos`
- Google 原生模式：`POST /v1beta/models/{model}:predictLongRunning`

Google 原生模式适用于直接传递 Veo 专有结构，例如 `instances[].referenceImages`、`parameters.aspectRatio`、`parameters.durationSeconds` 等。

## 参数

| 名称 | 位置 | 必填 | 描述 |
|---|---|---|---|
| model | path | True | Veo 模型名称。 |
