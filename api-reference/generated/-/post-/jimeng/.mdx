# 即梦视频生成

- Source: `relay.json`
- Method: `POST`
- Path: `/jimeng/`
- Operation ID: `createJimengVideo`

## 说明

即梦官方 API 格式的视频生成接口。

支持通过 Action 参数指定操作类型：
- `CVSync2AsyncSubmitTask`: 提交视频生成任务
- `CVSync2AsyncGetResult`: 获取任务结果

需要在查询参数中指定 Action 和 Version。


## 参数

| 名称 | 位置 | 必填 | 描述 |
|---|---|---|---|
| Action | query | True | API 操作类型 |
| Version | query | True | API 版本 |

## 请求体

```json
{'content': {'application/json': {'schema': {'type': 'object', 'description': '即梦官方 API 请求格式', 'properties': {'req_key': {'type': 'string', 'description': '请求类型标识'}, 'prompt': {'type': 'string', 'description': '文本描述'}, 'binary_data_base64': {'type': 'array', 'items': {'type': 'string'}, 'description': 'Base64 编码的图片数据'}}}, 'example': {'req_key': 'jimeng_video_generation', 'prompt': '一只猫在弹钢琴'}}}}
```

## 响应

| 状态码 | 说明 |
|---|---|
| 200 | 成功处理请求 |
| 400 | 请求参数错误 |
