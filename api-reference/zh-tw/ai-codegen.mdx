description: "本頁說明如何直接使用文件頁面的「複製頁面」功能，讓 AI 生成可執行的調用程式碼。"
---

## 建議做法

1. 打開目標介面頁，直接點擊「複製頁面」
2. 將複製得到的整頁內容整體發給 AI
3. 告訴 AI 你要使用的語言、業務參數和回傳處理方式
4. 要求 AI 輸出完整請求程式碼、錯誤處理，以及非同步任務查詢邏輯（若介面涉及任務查詢）
5. 回到文件頁或測試頁核對欄位名稱、路徑和鑑權方式是否一致

## 提示詞建議

```text
以下内容是我从 API 文档页直接复制的完整页面内容。
请基于这些内容生成可运行的调用代码。
要求：
1. 使用 Bearer Token 鉴权
2. Base URL 使用 https://api.aiid.edu.kg/v1
3. 保留完整请求体和错误处理
4. 用注释标明需要替换的 model、API Key 和业务参数
5. 如果这是长任务接口，请同时生成任务查询示例
```
