## 建議做法

1. 打開目標介面頁，直接點擊「複製頁面」
2. 將複製得到的整頁內容整體發給 AI
3. 告訴 AI 你要使用的語言、業務參數和回傳處理方式
4. 要求 AI 輸出完整請求程式碼、錯誤處理，以及非同步任務查詢邏輯（若介面涉及任務查詢）
5. 回到文件頁或測試頁核對欄位名稱、路徑和鑑權方式是否一致

## 提示詞建議

> 以下內容是我從 API 文件頁直接複製的完整頁面內容。
> 請根據這些內容產生可執行的呼叫程式碼。
> 要求：
> 1. 使用 Bearer Token 鑑權
> 2. Base URL 使用 https://api.aiid.edu.kg
> 3. 保留完整請求體和錯誤處理
> 4. 用註解標明需要替換的 model、API Key 和業務參數
> 5. 如果這是長任務介面，請同時產生任務查詢範例
