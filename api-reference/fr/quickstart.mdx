## Informations essentielles

- Base URL：`https://api.aiid.edu.kg`
- En-tête d'authentification : `Authorization: Bearer YOUR_API_KEY`
- Point de départ recommandé : `POST /v1/chat/completions`

## Exemple de requête minimale

```bash
curl https://api.aiid.edu.kg/v1/chat/completions   -H "Content-Type: application/json"   -H "Authorization: Bearer YOUR_API_KEY"   -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "hello"}]
  }'
```

## Éléments à vérifier avant d'accéder à d'autres interfaces

- Avez-vous remplacé correctement le Base URL ?
- Avez-vous inclus le Bearer Token ?
- Utilisez-vous le format de requête requis par la page de l'interface ?
- Avez-vous enregistré le `task_id` ou le `operation_name` nécessaire à la requête de tâche vidéo ?
