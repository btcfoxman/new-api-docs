# AI 生成调用代码

本页说明如何基于接口文档，让 AI 直接生成可运行的调用代码。

## 推荐做法

1. 复制目标接口页中的路径、请求体示例和响应结构
2. 告诉 AI 使用的语言、Base URL、鉴权方式和目标模型
3. 要求 AI 输出完整的请求代码和错误处理
4. 回到文档页或测试页核对字段名与路径是否一致

## 提示词建议

```text
请基于以下 API 文档生成可运行的调用代码。
要求：
1. 使用 Bearer Token 鉴权
2. Base URL 使用 https://api.aiid.edu.kg/v1
3. 保留完整请求体和错误处理
4. 用注释标明需要替换的 model、API Key 和业务参数
```
