---

## 推荐做法

1. 打开目标接口页，直接点击“复制页面”
2. 将复制得到的整页内容整体发给 AI
3. 告诉 AI 你要使用的语言、业务参数和返回处理方式
4. 要求 AI 输出完整请求代码、错误处理，以及异步任务查询逻辑（若接口涉及任务查询）
5. 回到文档页或测试页核对字段名、路径和鉴权方式是否一致

## 提示词建议

```text
以下内容是我从 API 文档页直接复制的完整页面内容。
请基于这些内容生成可运行的调用代码。
要求：
1. 使用 Bearer Token 鉴权
2. Base URL 使用 https://api.aiid.edu.kg/v1
3. 保留完整请求体和错误处理
4. 用注释标明需要替换的 model、API Key 和业务参数
5. 如果这是长任务接口，请同时生成任务查询示例
```
