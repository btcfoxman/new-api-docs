---
title: "即梦 (Jimeng)"
description: "即梦视频生成接口"
---

- Method: `POST`
- Path: `/jimeng/`

## 请求体字段

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| prompt | string | 是 | 视频内容描述 |
| duration | integer | 否 | 视频时长（秒） |
| ratio | string | 否 | 画幅比例，例如 `16:9` |

## 成功响应字段

| 字段 | 类型 | 说明 |
|---|---|---|
| task_id | string | 任务 ID |
| status | string | 初始状态 |
| created_at | string | 创建时间 |
