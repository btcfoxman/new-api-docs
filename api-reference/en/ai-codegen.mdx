description: "This page explains how to use the \"Copy Page\" feature on documentation pages to have AI generate executable integration code."
---

## Recommended Practices

1. Open the target API page and click "Copy Page."
2. Send the entire copied content to the AI.
3. Specify the programming language, business parameters, and response handling method to the AI.
4. Ask the AI to output complete request code, error handling, and asynchronous task polling logic (if the API involves task queries).
5. Return to the documentation or test page to verify that field names, paths, and authentication methods are consistent.

## Prompt Suggestions

```text
以下内容是我从 API 文档页直接复制的完整页面内容。
请基于这些内容生成可运行的调用代码。
要求：
1. 使用 Bearer Token 鉴权
2. Base URL 使用 https://api.aiid.edu.kg/v1
3. 保留完整请求体和错误处理
4. 用注释标明需要替换的 model、API Key 和业务参数
5. 如果这是长任务接口，请同时生成任务查询示例
```
