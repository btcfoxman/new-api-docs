## 推奨される方法

1. 対象のエンドポイントページを開き、「ページをコピー」を直接クリックします。
2. コピーしたページ内容をすべてAIに送信します。
3. 使用する言語、ビジネスパラメータ、およびレスポンスの処理方法をAIに伝えます。
4. 完全なリクエストコード、エラーハンドリング、および（エンドポイントにタスク照会が含まれる場合は）非同期タスク照会ロジックを出力するようAIに依頼します。
5. ドキュメントページまたはテストページに戻り、フィールド名、パス、認証方式が一致しているかを確認します。

## プロンプトのヒント

> 以下の内容は、API のドキュメントページから直接コピーしたページ全文です。
> これらの内容に基づいて、実行可能な呼び出しコードを生成してください。
> 要件:
> 1. Bearer Token 認証を使用する
> 2. Base URL には https://api.aiid.edu.kg を使用する
> 3. 完全なリクエストボディとエラーハンドリングを保持する
> 4. 置き換えが必要な model、API Key、および業務パラメータをコメントで明記する
> 5. これが長時間実行タスクのインターフェースである場合は、あわせてタスク照会の例も生成してください
