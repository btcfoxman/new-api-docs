## 推奨される方法

1. 対象のエンドポイントページを開き、「ページをコピー」を直接クリックします。
2. コピーしたページ内容をすべてAIに送信します。
3. 使用する言語、ビジネスパラメータ、およびレスポンスの処理方法をAIに伝えます。
4. 完全なリクエストコード、エラーハンドリング、および（エンドポイントにタスク照会が含まれる場合は）非同期タスク照会ロジックを出力するようAIに依頼します。
5. ドキュメントページまたはテストページに戻り、フィールド名、パス、認証方式が一致しているかを確認します。

## プロンプトのヒント

```text
以下内容是我从 API 文档页直接复制的完整页面内容。
请基于这些内容生成可运行的调用代码。
要求：
1. 使用 Bearer Token 鉴权
2. Base URL 使用 https://api.aiid.edu.kg/v1
3. 保留完整请求体和错误处理
4. 用注释标明需要替换的 model、API Key 和业务参数
5. 如果这是长任务接口，请同时生成任务查询示例
```
