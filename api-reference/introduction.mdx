## 协议与编码

- 协议：`HTTPS`
- 数据格式：`application/json`
- 字符编码：`UTF-8`

## 通用请求头

```http
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

## 通用响应结构

成功响应通常返回业务数据对象；失败响应返回 `error` 对象，包含 `code` 与 `message`。

## 幂等与重试

1. 对可重试错误（429/503）使用指数退避。
2. 视频生成等异步任务避免重复提交，可在业务层做幂等键控制。
