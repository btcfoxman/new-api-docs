# 在线调试

你可以直接在文档页的 API Playground 中发起请求，无需先写代码。

## 使用步骤

1. 打开目标接口页，点击 `试一试`。
2. 在页面内填写 `api_key`。该值只保存在当前浏览器本地，不会同步到服务器，也不会在团队成员之间共享。
3. 根据接口要求填写路径参数、查询参数或请求体。
4. 点击 `发送`，查看返回结果与错误信息。

## 调试建议

- 先确认 Base URL、接口路径和模型名称是否正确。
- 对长任务接口，优先记录返回的 `task_id` 或 `operation_name`，再调用查询接口。
- 若接口支持多个参数模式，优先使用文档示例请求体验证最小可用参数。
