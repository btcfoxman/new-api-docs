# AI 生成调用代码

你可以直接基于接口文档，让 AI 先生成一版可运行代码，再按你的业务参数做微调。

## 推荐流程

1. 复制目标接口的文档内容，包括接口路径、请求体示例、返回结构和补充说明。
2. 把文档内容贴给 GPT、Claude 或其他代码生成模型，并明确要求输出指定语言的完整调用代码。
3. 将生成的代码复制到本地项目中，替换 `Base URL`、`API Key`、模型名和业务参数。
4. 回到文档页或测试页，交叉验证请求路径、字段名和返回结构。

## 提示词建议

```text
请基于以下 API 文档生成 Python/Node.js/Go 调用代码。
要求：
1. 使用 Bearer Token 鉴权
2. Base URL 使用 https://api.aiid.edu.kg/v1
3. 保留完整请求体和错误处理
4. 用注释标明需要替换的 model、API Key 和业务参数
```

## 交付前检查

- 是否填写了正确的 Base URL
- 是否使用了正确的接口路径
- 是否带上了 `Authorization: Bearer YOUR_API_KEY`
- 是否按文档要求传了必填字段
