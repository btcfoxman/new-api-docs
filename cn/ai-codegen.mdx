## 推荐流程

1. 打开目标接口页，点击页面右上角或上下文操作中的“复制页面”。
2. 将复制得到的整页内容直接发给 GPT、Claude 或其他代码生成模型。
3. 只补充你的目标语言、业务参数和需要的返回处理方式，无需手工再整理接口路径、请求体和响应结构。
4. 将生成代码放回本地项目后，再回到文档页核对 Base URL、鉴权方式和必填字段。

## 提示词建议

> 以下内容是我从 API 文档页直接复制的完整页面内容。
> 请基于这些内容生成 Python/Node.js/Go 调用代码。
> 要求：
> 1. 使用 Bearer Token 鉴权
> 2. Base URL 使用 https://api.aiid.edu.kg
> 3. 保留完整请求体和错误处理
> 4. 用注释标明需要替换的 model、API Key 和业务参数
> 5. 若该接口是异步任务接口，补上查询任务结果的示例

## 交付前检查

- 是否填写了正确的 Base URL
- 是否使用了正确的接口路径
- 是否带上了 `Authorization: Bearer YOUR_API_KEY`
- 是否按文档要求传了必填字段
