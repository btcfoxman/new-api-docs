## 核心能力分布

### 1. 对话与推理 (Chat & Reasoning)
- **通用模型**：GPT-4o, Claude 3.5 Sonnet, Gemini 1.5 Pro。
- **推理模型**：OpenAI o1, DeepSeek V3/R1。
- **国产领先**：通义千问 (Qwen), 腾讯混元 (Hunyuan), 豆包 (Doubao)。

### 2. 多模态与创意 (Multimodal)
- **图像生成**：Midjourney, DALL·E 3, Flux Pro, Ideogram 2.0。
- **视频生成**：Luma Dream Machine, Sora, Kling, Kling 视频。
- **音频处理**：Suno v3.5 (音乐), Whisper (转录), OpenAI TTS (语音)。

## 接入规范

- **API Base URL**: `https://api.aiid.edu.kg/v1`
- **协议标准**: OpenAI RESTful API / SSE
- **鉴权方式**: Bearer Token (sk-...)
