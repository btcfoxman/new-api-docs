## Quy trình đề xuất

1. Mở trang API mục tiêu, nhấp vào "Sao chép trang" ở góc trên bên phải hoặc trong menu ngữ cảnh.
2. Gửi trực tiếp toàn bộ nội dung trang đã sao chép cho GPT, Claude hoặc các mô hình tạo mã khác.
3. Chỉ cần bổ sung ngôn ngữ mục tiêu, tham số nghiệp vụ và cách xử lý phản hồi mong muốn; không cần tự tay sắp xếp lại đường dẫn API, body yêu cầu và cấu trúc phản hồi.
4. Sau khi đưa mã đã tạo vào dự án local, hãy quay lại trang tài liệu để đối chiếu Base URL, phương thức xác thực và các trường bắt buộc.

## Gợi ý Prompt

```text
以下内容是我从 API 文档页直接复制的完整页面内容。
请基于这些内容生成 Python/Node.js/Go 调用代码。
要求：
1. 使用 Bearer Token 鉴权
2. Base URL 使用 https://api.aiid.edu.kg/v1
3. 保留完整请求体和错误处理
4. 用注释标明需要替换的 model、API Key 和业务参数
5. 若该接口是异步任务接口，补上查询任务结果的示例
```

## Kiểm tra trước khi bàn giao

- Đã điền đúng Base URL chưa
- Đã sử dụng đúng đường dẫn API chưa
- Đã kèm theo `Authorization: Bearer YOUR_API_KEY` chưa
- Đã truyền các trường bắt buộc theo yêu cầu của tài liệu chưa
