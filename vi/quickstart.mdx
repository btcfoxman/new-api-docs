## Bước 1: Lấy API Key

1. Đăng nhập vào [bảng điều khiển 出海营 API](https://aiid.edu.kg/console)
2. Truy cập trang Token để tạo API Key
3. Sau đó sử dụng thông qua `Authorization: Bearer YOUR_API_KEY`

## Bước 2: Cấu hình Base URL

```bash
API_BASE_URL="https://api.aiid.edu.kg/v1"
```

Nếu client hiện tại mặc định là:

- `https://api.openai.com`
- `https://api.openai.com/v1`

Thì hãy thay thế tương ứng bằng:

- `https://api.aiid.edu.kg`
- `https://api.aiid.edu.kg/v1`

## Bước 3: Gửi yêu cầu đầu tiên

### Python

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.aiid.edu.kg/v1",
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello, what can you do?"}],
)

print(response.choices[0].message.content)
```

### cURL

```bash
curl https://api.aiid.edu.kg/v1/chat/completions   -H "Content-Type: application/json"   -H "Authorization: Bearer YOUR_API_KEY"   -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello world!"}]
  }'
```

## Kiến nghị tích hợp

- Các dự án mới nên ưu tiên bắt đầu từ giao diện tương thích OpenAI
- Khi chỉnh sửa ảnh, cần đặc biệt lưu ý `multipart/form-data` và yêu cầu đối với trường tệp
- Với API tác vụ video, cần lưu trước `task_id` hoặc `operation_name` rồi mới truy vấn
- Khi gỡ lỗi thất bại, hãy ưu tiên kiểm tra mã trạng thái HTTP, `error.code` và `error.message`
