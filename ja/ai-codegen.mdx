## 推奨フロー

1. 対象の API ページを開き、ページ右上またはコンテキスト操作内の「ページをコピー」をクリックします。
2. コピーしたページ内容をそのまま GPT、Claude、またはその他のコード生成モデルに送信します。
3. ターゲット言語、ビジネスパラメータ、および必要なレスポンス処理方法を追記するだけで済みます。API パス、リクエストボディ、レスポンス構造を手動で整理する必要はありません。
4. 生成されたコードをローカルプロジェクトに配置した後、ドキュメントページに戻って Base URL、認証方式、および必須フィールドを再確認してください。

## プロンプトのヒント

> 以下の内容は、API のドキュメントページから直接コピーしたページ全文です。
> これらの内容に基づいて、Python/Node.js/Go の呼び出しコードを生成してください。
> 要件:
> 1. Bearer Token 認証を使用する
> 2. Base URL には https://api.aiid.edu.kg/v1 を使用する
> 3. 完全なリクエストボディとエラーハンドリングを保持する
> 4. 置き換えが必要な model、API Key、および業務パラメータをコメントで明記する
> 5. このインターフェースが非同期タスク用のインターフェースである場合は、タスク結果を照会するサンプルを追加する

## デプロイ前のチェック

- 正しい Base URL が入力されているか
- 正しい API パスが使用されているか
- `Authorization: Bearer YOUR_API_KEY` が含まれているか
- ドキュメントの要件に従って必須フィールドが渡されているか
