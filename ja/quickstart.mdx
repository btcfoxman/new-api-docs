## ステップ 1: API キーの取得

1. [出海营 API コンソール](https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D)にログインします
2. トークンページに移動し、API キーを作成します
3. その後、https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D を通じて使用します

## ステップ 2: Base URL の設定

```bash
API_BASE_URL="https://api.aiid.edu.kg/v1"
```

既存のクライアントでデフォルトが以下のように記述されている場合：

- `https://api.openai.com`
- `https://api.openai.com/v1`

それぞれ次のように置き換えます：

- `https://api.aiid.edu.kg`
- `https://api.aiid.edu.kg/v1`

## ステップ 3: 最初のリクエストの送信

### Python

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.aiid.edu.kg/v1",
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "你好，请给我一个快速接入建议"}],
)

print(response.choices[0].message.content)
```

### cURL

```bash
curl https://api.aiid.edu.kg/v1/chat/completions   -H "Content-Type: application/json"   -H "Authorization: Bearer YOUR_API_KEY"   -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello world!"}]
  }'
```

## 導入のアドバイス

- 新規プロジェクトでは OpenAI 互換インターフェースの使用を優先してください
- 画像編集では https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D とファイルフィールドの要件に注意する必要があります
- 動画タスクインターフェースでは、まず https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D または `operation_name` を保存してからクエリを実行してください
- デバッグに失敗した場合は、HTTP ステータスコード、https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D、`error.message` を優先的に確認してください
