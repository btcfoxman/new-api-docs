---
title: "Error Codes"
description: "Common errors and how to resolve"
---

| HTTP | Meaning | Action |
|---|---|---|
| 400 | Invalid request | Validate required fields |
| 401 | Unauthorized | Check API key |
| 403 | Forbidden | Check model permission |
| 404 | Not found | Check endpoint or task id |
| 429 | Rate limited | Reduce concurrency and retry |
| 500 | Server error | Retry with request id logged |
