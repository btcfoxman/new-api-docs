# new-api 开发者文档

`new-api` 提供统一网关，帮助你使用一致的鉴权与请求方式接入多种 AI 能力。

## 核心价值

- 单一 Base URL，减少多平台 SDK 切换成本
- 统一 API Key 鉴权，降低接入复杂度
- 面向文本、图像、视频、多模态场景的一致化调用体验

## 能力矩阵

| 能力类型 | 代表接口 | 适用场景 |
|---|---|---|
| 文本/对话 | OpenAI Chat、Claude Messages | 问答、Agent、内容生成 |
| 图像生成 | Qwen 图像、即梦视频生成入口 | 文生图、图像编辑、创意生成 |
| 视频生成 | Sora、Kling 文生视频、Kling 图生视频 | 短视频内容生产 |
| 多模态/其他 | Embeddings、Rerank、Realtime | 检索增强、排序、实时交互 |
