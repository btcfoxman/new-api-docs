## 推薦流程

1. 開啟目標介面頁，點擊頁面右上角或上下文操作中的「複製頁面」。
2. 將複製得到的整頁內容直接發給 GPT、Claude 或其他程式碼生成模型。
3. 只需補充您的目標語言、業務參數和需要的內容處理方式，無需手動再整理介面路徑、請求體和回應結構。
4. 將產生的程式碼放回在地專案後，再回到文件頁核對 Base URL、鑒權方式和必填欄位。

## 提示詞建議

```text
以下内容是我从 API 文档页直接复制的完整页面内容。
请基于这些内容生成 Python/Node.js/Go 调用代码。
要求：
1. 使用 Bearer Token 鉴权
2. Base URL 使用 https://api.aiid.edu.kg/v1
3. 保留完整请求体和错误处理
4. 用注释标明需要替换的 model、API Key 和业务参数
5. 若该接口是异步任务接口，补上查询任务结果的示例
```

## 交付前檢查

- 是否填寫了正確的 Base URL
- 是否使用了正確的介面路徑
- 是否帶上了 `Authorization: Bearer YOUR_API_KEY`
- 是否按文件要求傳送了必填欄位
