## 第一步：獲取 API Key

1. 登入 [出海營 API 主控台](https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D)
2. 進入權杖頁面建立 API Key
3. 後續透過 https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D 使用

## 第二步：設定 Base URL

```bash
API_BASE_URL="https://api.aiid.edu.kg/v1"
```

如果現有用戶端預設寫法為：

- `https://api.openai.com`
- `https://api.openai.com/v1`

則分別替換為：

- `https://api.aiid.edu.kg`
- `https://api.aiid.edu.kg/v1`

## 第三步：發起第一個請求

### Python

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.aiid.edu.kg/v1",
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "你好，请给我一个快速接入建议"}],
)

print(response.choices[0].message.content)
```

### cURL

```bash
curl https://api.aiid.edu.kg/v1/chat/completions   -H "Content-Type: application/json"   -H "Authorization: Bearer YOUR_API_KEY"   -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello world!"}]
  }'
```

## 接入建議

- 新專案優先從 OpenAI 相容介面開始
- 圖片編輯必須注意 https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D 與檔案欄位要求
- 影片任務介面應先儲存 https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D 或 `operation_name` 再查詢
- 偵錯失敗時優先查看 HTTP 狀態碼、https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D 和 `error.message`
