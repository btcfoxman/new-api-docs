## 模型分組

### Nano Banana

主要用於圖像生成與編輯，也支援 Gemini 原生格式接入。



| 模型 | 主要接入方式 |

|---|---|

| `nano-banana` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2-2k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2-4k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro-2k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro-4k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |



### Sora

優先走 `/v1/videos` OpenAI 影片相容格式，適合統一影片接入。



| 模型 | 主要接入方式 |

|---|---|

| `sora-2` | `/v1/videos` |

| `sora-2-pro` | `/v1/videos` |



### Gemini Omni

`gemini-omni` 使用 new-api 對外公開模型名，可透過 `/v1/videos` 或 `/api/v3/contents/generations/tasks` 呼叫；使用 `mode=t2v/r2v/edit` 區分文生、參考生成和影片編輯，時長會自動匹配到 4 / 6 / 8 / 10 秒檔。



| 模型 | 主要接入方式 |

|---|---|

| `gemini-omni` | `/v1/videos` / `/api/v3/contents/generations/tasks` |



### HappyHorse

支援阿里 DashScope 官方影片生成格式，也支援透過 `/v1/videos` 相容接入。



| 模型 | 主要接入方式 |

|---|---|

| `happyhorse-1.0` | `/api/v1/services/aigc/video-generation/video-synthesis` / `/v1/videos` |

| `happyhorse-1.0-i2v` | `/api/v1/services/aigc/video-generation/video-synthesis` / `/v1/videos` |

| `happyhorse-1.0-t2v` | `/api/v1/services/aigc/video-generation/video-synthesis` / `/v1/videos` |

| `happyhorse-1.0-r2v` | `/api/v1/services/aigc/video-generation/video-synthesis` / `/v1/videos` |

| `happyhorse-1.0-video-edit` | `/api/v1/services/aigc/video-generation/video-synthesis` / `/v1/videos` |



### Veo

Veo 主模型優先走 `/v1/videos` 相容格式；`/v1beta/...` 為 Veo 原生協議預留說明。`gemini-omni` 不走 Google 原生路徑，請使用 `/v1/videos` 或 Seedance 任務格式。



| 模型 | 主要接入方式 |

|---|---|

| `veo3` | `/v1/videos` / `/v1beta/...` |

| `veo3-fast` | `/v1/videos` / `/v1beta/...` |

| `veo3-fast-frames` | `/v1/videos` / `/v1beta/...` |

| `veo3-frames` | `/v1/videos` / `/v1beta/...` |

| `veo3-pro` | `/v1/videos` / `/v1beta/...` |

| `veo3-pro-frames` | `/v1/videos` / `/v1beta/...` |

| `veo3.1` | `/v1/videos` / `/v1beta/...` |

| `veo3.1-4k` | `/v1/videos` / `/v1beta/...` |

| `veo3.1-components` | `/v1/videos` / `/v1beta/...` |

| `veo3.1-fast` | `/v1/videos` / `/v1beta/...` |

| `veo3.1-pro` | `/v1/videos` / `/v1beta/...` |

| `veo3.1-pro-4k` | `/v1/videos` / `/v1beta/...` |



### Grok

目前公開的是影片生成模型，建議優先按統一影片介面接入。



| 模型 | 主要接入方式 |

|---|---|

| `grok-imagine-1.0-video` | `/v1/videos` |

| `grok-video-3` | `/v1/videos` |



### Seedance

支援 `/api/v3/contents/generations/tasks` 原生格式，也支援透過 `/v1/videos` 相容接入；`gemini-omni` 可重用該任務格式並用 `mode=t2v/r2v/edit` 區分能力。



| 模型 | 主要接入方式 |

|---|---|

| `doubao-seedance-1-0-lite-i2v-250428` | `/api/v3/contents/generations/tasks` / `/v1/videos` |

| `doubao-seedance-1-0-lite-t2v-250428` | `/api/v3/contents/generations/tasks` / `/v1/videos` |

| `doubao-seedance-1-0-pro-250528` | `/api/v3/contents/generations/tasks` / `/v1/videos` |

| `doubao-seedance-1-0-pro-fast-251015` | `/api/v3/contents/generations/tasks` / `/v1/videos` |

| `doubao-seedance-1-5-pro-251215` | `/api/v3/contents/generations/tasks` / `/v1/videos` |

| `doubao-seedance-2-0-260128` | `/api/v3/contents/generations/tasks` / `/v1/videos` |

| `doubao-seedance-2-0-fast-260128` | `/api/v3/contents/generations/tasks` / `/v1/videos` |



### Kling Images

透過 `/v1/images/generations` 對外提供 Kling 圖片生成、Omni 圖片和虛擬試衣能力。



| 模型 | 主要接入方式 |

|---|---|

| `kling-image` | `/v1/images/generations` |

| `kling-omni-image` | `/v1/images/generations` |

| `kolors-virtual-try-on-v1` | `/v1/images/generations` |

| `kolors-virtual-try-on-v1-5` | `/v1/images/generations` |



### Kling Videos

同時公開模型專用介面與統一影片入口。



| 模型 | 主要接入方式 |

|---|---|

| `kling-video` | `/kling/v1/...` / `/v1/videos` |

| `kling-*` | `/kling/v1/...` / `/v1/videos` |



### Hailuo

使用統一影片能力接入，適合文字生成影片場景。



| 模型 | 主要接入方式 |

|---|---|

| `hailuo*` | `/v1/videos` |



### Vidu

支援 Vidu 官方影片生成格式，也支援透過 `/v1/videos` 相容接入，適合文生、單圖、首尾幀和多參考圖場景。



| 模型 | 主要接入方式 |

|---|---|

| `viduq3-turbo` | `/ent/v2/...` / `/v1/videos` |

| `vidu*` | `/ent/v2/...` / `/v1/videos` |
