## 模型分組

### Nano Banana

主要用於圖像生成與編輯，也支援 Gemini 原生格式接入。



| 模型 | 主要接入方式 |

|---|---|

| `nano-banana` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2-2k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2-4k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro-2k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro-4k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |



### Sora

優先走 https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D OpenAI 影片相容格式，適合統一影片接入。



| 模型 | 主要接入方式 |

|---|---|

| `sora-2` | `/v1/videos` |

| `sora-2-pro` | `/v1/videos` |

|---|---|

| `grok-video-3` | `/v1/videos` |

| `doubao-seedance-2-0-fast-260128` | `/api/v3/contents/generations/tasks` / `/v1/videos` |



### Kling

同時公開模型專用介面與統一影片入口。



| 模型 | 主要接入方式 |

|---|---|

| `kling-video` | `/kling/v1/...` / `/v1/videos` |

| `kling-*` | `/kling/v1/...` / `/v1/videos` |



### Hailuo

使用統一影片能力接入，適合文字生成影片場景。



| 模型 | 主要接入方式 |

|---|---|

| `hailuo*` | `/v1/videos` |



### Vidu

以統一影片能力說明為主，適合單圖、首尾幀和多參考圖場景。



| 模型 | 主要接入方式 |

|---|---|

| `vidu*` | `/v1/videos` |
