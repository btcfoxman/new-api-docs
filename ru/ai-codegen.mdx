## Рекомендуемый процесс

1. Откройте страницу целевого API и нажмите «Копировать страницу» в правом верхнем углу или в контекстном меню.
2. Отправьте скопированное содержимое страницы напрямую в GPT, Claude или другую модель генерации кода.
3. Укажите только целевой язык, бизнес-параметры и требуемую логику обработки ответа; нет необходимости вручную описывать путь API, тело запроса и структуру ответа.
4. После того как сгенерированный код будет возвращён в локальный проект, снова откройте страницу документации и проверьте Base URL, способ аутентификации и обязательные поля.

## Рекомендации по промптам

> Ниже приведено полное содержимое страницы, которое я напрямую скопировал со страницы документации API.
> Пожалуйста, на основе этих материалов сгенерируйте код вызова на Python/Node.js/Go.
> Требования:
> 1. Использовать аутентификацию Bearer Token
> 2. Base URL использует https://api.aiid.edu.kg
> 3. Сохранить полное тело запроса и обработку ошибок
> 4. В комментариях отметьте model, API Key и бизнес-параметры, которые нужно заменить
> 5. Если этот интерфейс является интерфейсом асинхронных задач, добавьте пример запроса результата задачи

## Проверка перед использованием

- Указан ли правильный Base URL
- Используется ли верный путь API
- Передан ли параметр `Authorization: Bearer YOUR_API_KEY`
- Переданы ли все обязательные поля согласно документации
