## Группировка моделей

|---|---|

| `nano-banana` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2-2k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro-4k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

Через `/v1/images/generations` предоставляются возможности генерации изображений Kling, Omni Image и виртуальной примерки.



| Модель | Основной способ подключения |

|---|---|

| `kling-image` | `/v1/images/generations` |

| `kling-omni-image` | `/v1/images/generations` |

| `kolors-virtual-try-on-v1` | `/v1/images/generations` |

| `kolors-virtual-try-on-v1-5` | `/v1/images/generations` |



### Kling Videos

Одновременно открыты специализированный интерфейс модели и унифицированный вход для видео.



| Модель | Основной способ подключения |

|---|---|

| `kling-video` | `/kling/v1/...` / `/v1/videos` |

| `kling-*` | `/kling/v1/...` / `/v1/videos` |



### Hailuo

Подключение с использованием унифицированных возможностей видео, подходит для сценариев генерации видео из текста.



| Модель | Основной способ подключения |

|---|---|

| `hailuo*` | `/v1/videos` |



### Vidu

Основное внимание уделено описанию унифицированных возможностей видео, подходит для сценариев с одним изображением, начальным/конечным кадрами и несколькими референсными изображениями.



| Модель | Основной способ подключения |

|---|---|

| `vidu*` | `/v1/videos` |
