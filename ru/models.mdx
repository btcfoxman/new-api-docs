## Группировка моделей

| `nano-banana-2` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2-2k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-2-4k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro-2k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |

| `nano-banana-pro-4k` | `/v1/images/generations` / `/v1/images/edits` / `/v1beta/...` |



### Sora

Предпочтительно использовать `/v1/videos` OpenAI совместимый видеоформат, подходящий для унифицированного подключения видео.



| Модель | Основной способ подключения |

|---|---|

| `sora-2` | `/v1/videos` |

| `sora-2-pro` | `/v1/videos` |



### Gemini Omni

`gemini-omni` использует new-api для публичного имени модели; вызывать можно через `/v1/videos` или `/api/v3/contents/generations/tasks`; `mode=t2v/r2v/edit` используется для различения text-to-image, генерации по референсу и видеомонтажа, а длительность автоматически подбирается к вариантам 4 / 6 / 8 / 10 секунд.



| Модель | Основной способ подключения |

|---|---|

| `gemini-omni` | `/v1/videos` / `/api/v3/contents/generations/tasks` |

### Hailuo

Подключение с использованием унифицированных возможностей видео, подходит для сценариев генерации видео из текста.



| Модель | Основной способ подключения |

|---|---|

| `hailuo*` | `/v1/videos` |



### Vidu

Поддерживается официальный формат генерации видео Vidu, а также совместимое подключение через `/v1/videos`; подходит для сценариев text-to-video, одного изображения, первого и последнего кадра и нескольких референсных изображений.



| Модель | Основной способ подключения |

|---|---|

| `viduq3-turbo` | `/ent/v2/...` / `/v1/videos` |

| `vidu*` | `/ent/v2/...` / `/v1/videos` |
