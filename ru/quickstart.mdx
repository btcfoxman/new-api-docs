## Шаг 1: Получение ключа API

1. Войдите в [консоль 出海营 API](https://aiid.edu.kg/console)
2. Перейдите на страницу токенов, чтобы создать ключ API
3. В дальнейшем используйте через `Authorization: Bearer YOUR_API_KEY`

## Шаг 2: Настройка Base URL

```bash
API_BASE_URL="https://api.aiid.edu.kg"
```

Если в существующем клиенте по умолчанию указано:

- `https://api.openai.com`
- `https://api.openai.com/v1`

То замените их соответственно на:

- `https://api.aiid.edu.kg`
- `https://api.aiid.edu.kg`

## Шаг 3: Выполнение первого запроса

### Python

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.aiid.edu.kg",
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello, what can you do?"}],
)

print(response.choices[0].message.content)
```

### cURL

```bash
curl https://api.aiid.edu.kg/v1/chat/completions   -H "Content-Type: application/json"   -H "Authorization: Bearer YOUR_API_KEY"   -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello world!"}]
  }'
```

## Рекомендации по интеграции

- Для новых проектов рекомендуется начинать с интерфейсов, совместимых с OpenAI
- При редактировании изображений обязательно учитывайте требования к `multipart/form-data` и полям файла
- Для интерфейса видеозадач сначала сохраните `task_id` или `operation_name`, а затем выполняйте запрос
- Если отладка не удалась, в первую очередь проверьте код состояния HTTP, `error.code` и `error.message`
