---
title: "错误码手册"
description: "统一错误码与处理建议"
---

| HTTP 状态码 | 含义 | 建议处理 |
|---|---|---|
| 400 | 请求参数不合法 | 检查必填字段、类型与取值范围 |
| 401 | 鉴权失败 | 检查 API Key 是否正确与是否过期 |
| 403 | 无权限访问 | 检查账号权限或模型访问范围 |
| 404 | 资源不存在 | 检查路径、任务 ID、模型 ID |
| 429 | 触发频率限制 | 降低并发并使用指数退避重试 |
| 500 | 服务内部错误 | 记录请求 ID，稍后重试或联系支持 |
| 503 | 服务暂不可用 | 短暂等待后重试 |

## 错误响应示例

```json
{
  "error": {
    "code": "invalid_request",
    "message": "model is required",
    "request_id": "req_xxx"
  }
}
```
