# 欢迎使用 new-api

我们的 API 服务提供了访问全球顶级 AI 模型的能力，包括 GPT-4o, Claude 3.5 Sonnet, Gemini 1.5 Pro 以及各种国产领先大模型。通过统一的 API 地址，您可以极速完成多个厂商的集成。

## 核心特性

- **多模型支持**：支持 OpenAI, Anthropic, Google, 百度, 阿里等 50+ 厂商模型。
- **协议兼容**：完全兼容 OpenAI 协议，大部分应用只需修改 Base URL 和 API Key 即可快速接入。
- **高可用性**：多机房冗余部署，确保全球开发者访问的极致稳定性。
- **透明定价**：采用标准计费体系，让每一笔模型调用清晰透明。
