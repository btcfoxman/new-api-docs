## Step 1: Obtain the API Key

1. Log in to the [Chuhaiying API Console](https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D)
2. Go to the Tokens page to create a API Key
3. Use it via https://www.google.com/search?q=%5B%5BKEEP_0%5D%5D in subsequent steps

## Step 2: Configure Base URL

```bash
API_BASE_URL="https://api.aiid.edu.kg/v1"
```

If the existing client default is set to:

- `https://api.openai.com`
- `https://api.openai.com/v1`

Then replace them with:

- `https://api.aiid.edu.kg`
- `https://api.aiid.edu.kg/v1`

## Step 3: Send your first request

### Python

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.aiid.edu.kg/v1",
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "你好，请给我一个快速接入建议"}],
)

print(response.choices[0].message.content)
```

### cURL

```bash
curl https://api.aiid.edu.kg/v1/chat/completions   -H "Content-Type: application/json"   -H "Authorization: Bearer YOUR_API_KEY"   -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello world!"}]
  }'
```
